## After setup.bat Completes

### 1. Configure Azure OpenAI
Edit `.env.local` with your details:
```
AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com
AZURE_OPENAI_GPT4_VISION_DEPLOYMENT=gpt-4-vision
AZURE_OPENAI_GPT4_DEPLOYMENT=gpt-4
AZURE_OPENAI_EMBEDDING_DEPLOYMENT=text-embedding-3-large
```

### 2. Azure Authentication
```cmd
az login
```

### 3. Assign RBAC Permissions
```cmd
az role assignment create --assignee YOUR_EMAIL --role "Cognitive Services OpenAI User" --scope /subscriptions/SUB_ID/resourceGroups/RG/providers/Microsoft.CognitiveServices/accounts/OPENAI_NAME
```

### 4. Add LBBD GeoJSON
Place your boundary file at:
```
public\geojson\lbbd-boundary.json
```

### 5. Start Development Server
```cmd
npm run dev
```

### 6. Open Browser
Navigate to: **http://localhost:3000**
